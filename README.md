# Mutated

A mod that adds potion of mutation and various mutations to Minecraft. Inspired and somewhat based on [Dungeon Crawl Stone Soup](https://crawl.develz.org/).

## List of Mutations
### Good Mutations
* Rapid Learner (`experience_boost`)
